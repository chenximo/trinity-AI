## 5.31 内测门禁（平台面 · P0）

| 优先级 | 模块 | 门禁 |
|--------|------|------|
| **P0** | 生文 API + 流式 | 文档示例 `POST /v1/chat/completions` **真通**（含 stream） |
| **P0** | 鉴权 | 无效 Key 401；有效 Key 可调通 |
| **P0** | 统一 API 基座 | 基址、模型 ID、错误体与文档一致 |
| **P0** | 计量 | 调用产生 `usage_event`（可查，可不自动扣费） |
| P1 | 限流 | 基础按 Key 限流 |
| P2 | 多模态 | 至少一条生图或生视频通路演示 |
| **不做** | Fallback 全链 | 6.30 前不阻塞内测 |
