## 完成度 % 与符号

- **子能力当前 %**：该条能力「实际做到什么」的主观验收（0 / 25 / 50 / 75 / 100），在分页 MD 维护。  
- **模块当前 %** = 该模块子项汇总；总览表写在 **% 前**：✅ 🟡 ⬜ ➖（见 [用户面-00-总览.md](./用户面-00-总览.md) 列说明）。  
- **5.31 目标 %**：内测验收线；前缀 ✅ = 须达标，🟡 = 允许精简。  
- **6.30 商用 %**：商用交付线；前缀 ✅ = 对标 OR 须交付，🟡 = 精简版（如 Chat 70%）。

## 与 PPT 同步

1. 每周只改各 MD 中的 **当前 %** 与「已完成 / 未完成」备注。  
2. PPT：**用户面** · **平台面** · **运营面** 各贴对应 `*-00-总览.md` 总表。  
3. 本周攻坚模块：再贴对应 `*-0x` 分页摘要。
