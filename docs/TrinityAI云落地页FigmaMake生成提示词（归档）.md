---

## 1. 产品与受众

- **产品名**：Trinity AI 云（副品牌 **Trinity** 顶栏展示）。
- **定位**：企业级 **多云 AI 算力纳管** — 统一接入腾讯云、华为云、阿里云及全球主流公有云账号与 GPU / 推理资源池；算力编排、弹性配额、成本分账、合规审计。
- **受众**：企业 IT、平台团队、采购与架构负责人。
- **调性**：可信、专业、偏 **SaaS / 云厂商控制台** 美学；避免游戏感或过度插画。

---

## 2. 画布与布局规则

- **主画布**：桌面 **1440px** 宽（可兼容 1280–1920），垂直滚动单页。
- **内容最大宽度**：正文区块约 **1180px**（`home-shell`）；顶栏与页脚为 **全宽**，左右 **50px** gutter（与内页一致）。
- **区块水平内边距**：主栏内正文相对视口左右各约 **15%** `padding-inline`（窄屏时可降为 24px 量级）。
- **顶栏高度**：**60px**，`sticky` 置顶；背景 **半透明白 + 背景模糊**（glass），底 **1px** 浅灰描边。

## 4. 页面结构（自上而下，须按顺序生成）

### A. 顶栏 `header`

- **左**：Logo — 圆角方标 **32×32**，皇家蓝底白星形图标 + 文案 **「Trinity」**（字重 700）。
- **中**（≥900px 显示）：导航  
  - 下拉 **「AI 云 ▾」** → 面板内两项：「AI 云能力总览」「多云对接与各云说明」  
  - 文字链 **「Trinity AI」**
- **右**：「中 / EN」小按钮（描边）· **注册**（幽灵蓝）· **登录**（渐变实心）· 汉堡（仅移动端，桌面隐藏）。
- 移动端：展开 **抽屉** 列出同上锚点 + 登录/注册。

### B. 首屏 Hero `section.home-banner`

- **背景**：极浅灰白竖向渐变；顶部 **径向淡蓝辉光**；上层 **淡蓝网格线**（约 48px 间距，低透明度），可暗示轻微动感。
- **布局**：宽屏 **两列网格** — 左文案右「控制台预览卡」；窄屏单列居中。
- **Eyebrow**：胶囊标签，浅蓝底 + 蓝点 + 文案：**「企业级多云 AI 算力与服务」**。
- **H1**：主标题 **「Trinity AI 云」** — 整段标题使用 **品牌蓝**（实作：标题置于强调样式内，与深灰副文案形成对比）。
- **副标题**（≤2 行桌面）：  
  *「纳管腾讯云、华为云、阿里云及全球主流公有云账号与 GPU / 推理资源池，提供算力编排、弹性配额、成本分账与合规审计能力，助力企业专注上云与用算力。」*
- **CTA 行**（三枚横排、统一高度约 44px）：  
  1. 渐变实心：**「立即咨询」**（锚 `#consult`）  
  2. 浅线框：**「查看折扣」**（锚 `#promo`）  
  3. 默认实心描边弱：**「云产品说明」**（锚 `#cloud-solutions`）
- **统计胶囊**（4 个横排小 chip）：  
  - `12+` 公有云区域接入 · `99.95%` 算力池可用性目标 · `5K+` GPU 卡时/日 · `8` 自建与托管地域节点  
- **能力 Pills**（描边小标签）：多云账号纳管 · GPU 弹性配额 · 专线/VPC 互联 · 账单分账与审计  
- **右侧预览卡** `home-preview-card`：白卡片带阴影；顶栏「云控制台 · 资源概览」+ 角标「同步」；内为 **3 个 KPI 数字**（运行中 GPU 节点、池化利用率峰值、本月云消费）+ **抽象条形图** + **三行列表**（集群健康、伸缩组状态、同步任务）。整体像 **监控台缩略图**，非真实数据界面。

### C. 重叠四宫格 `home-overlap`（负 margin 上提、压在 Hero 下缘的条带）

- 全宽条，内 **2×2 或 4 列** 链接卡片（白底、细边框、hover 上浮感）：  
  1. **各云对接说明** — 副文：主流公有云按标签查看对接能力与交付边界  
  2. **架构与合规** — 副文：接入、编排、成本与审计分层设计  
  3. **商务折扣** — 副文：算力券与年度框架  
  4. **上云与交付** — 副文：迁移评估、网络规划与专人交付  

### D. 多云厂商 Tabs `section#cloud-solutions`

- **小标题**：**「已支持对接的主流公有云」**
- **Tab 条**：腾讯云 · 华为云 · 阿里云 · Google Cloud · AWS（pill/下划线选中态，默认选中腾讯云）。
- **Tab 面板**（每云一屏，内容区为 **左文右图** 或 **上图下文**）：  
  - 左侧：该云 **场景标题**、要点列表、合规/网络关键词标签。  
  - 右侧：**宽屏装饰性 SVG/插图**（控制台、波形、仪表盘隐喻；腾讯云稿偏青绿深色控制台风）。  
- 五个 Tab 内容主题需 **视觉区分**（色点或左侧竖条），但 **组件结构统一**。

### E. AI 云服务 `section#cloud`

- **H2**：「AI 云服务」  
- **Lead**：一段企业向说明（多云统一纳管 AI 推理与 GPU、配额调度、成本分账、合规留痕）。  
- **四卡网格**（图标 + H3 + 短段落）：  
  1. 多云纳管 · 2. GPU 与弹性算力 · 3. 云成本与分账 · 4. 安全与合规  

### F. 架构优势 `section#arch`（交替灰底 `alt`）

- **H2**：「我们的架构优势」  
- **Lead**：一句分层设计说明。  
- **左**：竖向 **三层架构示意图**（三行圆角条 + 向下箭头）：公有云账号与网络接入层 → 资源编排与 GPU 算力池层 → 成本配额与合规审计层。  
- **右**：高可用 / 可审计 / 弹性扩展 **三段加粗说明**。  
- **下方三列 highlight**：多活容灾 · 成本可视 · 企业集成（每块：粗标题 + 2 行内描述）。

### G. 限时优惠 `section#promo`

- **H2**：「限时优惠 / 折扣活动」  
- **Lead**：权益以合同与公示为准。  
- **两张大 promo 卡**（一浅一深 slate）：  
  1. 新用户首月抵扣 + 角标「限时礼遇」+ CTA「联系销售」  
  2. 企业年度框架 + 角标「企业专享」+ CTA「获取报价单」  

### H. 咨询表单 `section#consult`（交替灰底）

- **H2**：「获取专属咨询」  
- **Lead**：1～2 个工作日联系。  
- **表单**：姓名 · 工作邮箱 · 公司/团队（选填）· 需求简述（多行）· 底部渐变 **「提交咨询」**。

### I. 页脚 `footer`

- **四列**：品牌简介（Trinity AI 云 + 一句定位）· 本页产品锚点列表 · 文档中心链接 · 合规与支持（隐私/条款/白皮书占位）。  
- **底栏**：© 2026 Trinity. 保留所有权利。

## 6. 交互与响应（标注即可）

- 顶栏 sticky；锚点平滑滚动。  
- **断点**：约 **900px** 切换顶栏主导航与抽屉；约 **960px** Hero 双列。  
- Tab、下拉、按钮需 **focus/hover** 状态（蓝浅底或描边加深）。

- 无需与 `index.html` 产品页完全像素一致，但 **色板与顶栏气质须一致**。
