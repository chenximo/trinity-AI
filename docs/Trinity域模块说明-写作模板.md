## 模板之外：建议补充的块（按需）

下列 **不必每个模块都写全**；有则填，无则删节标题。

1. **术语表**：域内专有名词（如「角色 / preset / 对比」）中英文与含义一行。  
2. **权限与可见性**（后台域尤其）：角色、菜单、按钮级可见规则。  
3. **埋点 / 分析**（若原型也要占位）：事件名列表。  
4. **国际化**：本模块是否首期只做中文、文案 key 约定。  
5. **性能与数据量级假设**：列表最大条数、是否需分页/虚拟列表（原型可写「正式版再接」）。  
6. **安全与合规提示**：敏感字段不落 Mock、脱敏示例。  
7. **与 Cursor Skill 的关系**：本模块 **不单独建 skill** 除非同一规则已对 AI 重复说明多次；域内纪律引用 `trinity-vue-prototype-monorepo` 与 `docs/Trinity前端Vue与Monorepo工程方案.md`。
