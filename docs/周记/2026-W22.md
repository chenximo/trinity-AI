### 备注（挑战性 / 卡点）

| 项 | 说明 |
| --- | --- |
| **模型上架节奏** | 表格维护、验收测试、运营后台配置需三线同步；单款模型联调耗时不定，批量上架需预留缓冲。 |
| **6 月二期范围** | 商用能力、计费与 SLA 未拍板前，文档与对外口径只能写「规划中」；**依赖产品决策会**。 |
| **内部应用对接** | 鉴权方式（Key / OAuth）、环境隔离、配额与审计要求若未定，对接说明只能写草案。 |
| **验收台 vs 真实网关** | 用例与报告目前基于 localhost / 样例 model；网关 URL、model 枚举、流式字段未定时，报告只能作内部联调参考，**不宜对外宣称「已验收通过」**。 |
| **FormSubmit 咨询发信** | 原型期走第三方表单投递，正式环境需自有后端或企业邮箱 API，并处理 spam / 白名单。 |
| **AI 云首页迭代频繁** | 轨道组件增删、配图多轮替换，说明视觉定稿仍在进行中；部署前建议冻结一版供产品签字。 |
