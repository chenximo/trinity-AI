# 云厂商 Logo 素材（可选）

业务页使用 **`CloudVendorLogo.vue`**，各厂商引用本目录 PNG（透明底官方图形标）。

其它厂商若需替换，可放入此目录并在组件中引用：

| 文件名 | 云厂商 |
|--------|--------|
| `aliyun.png` | 阿里云 |
| `tencent.png` | 腾讯云 |
| `huawei.png` | 华为云 |
| `aws.png` | AWS |
| `gcp.png` | Google Cloud |

**已接入**：`aliyun.png`、`tencent.png`、`huawei.png`、`aws.png`、`gcp.png`、`azure.png`  
建议：正方形、透明底；边长约 64–640px。
