# 云厂商 Logo 素材（可选）

业务页使用 **`CloudVendorLogo.vue`** 内嵌 SVG（透明底，适配 Tab / 浅色背景）。

若需替换为自有 PNG / SVG，可放入此目录备用（组件当前未读取目录文件）：

| 文件名 | 云厂商 |
|--------|--------|
| `aliyun.png` | 阿里云 |
| `tencent.png` | 腾讯云 |
| `huawei.png` | 华为云 |
| `aws.png` | AWS |
| `gcp.png` | Google Cloud |

建议：正方形、透明底；边长约 64–128px。
