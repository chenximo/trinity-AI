# 多模态概述

Trinity 支持在统一 API 下处理**图片、视频、音频与 PDF** 等多模态输入与生成能力。

::: tip
模型标识使用 `provider/model` 格式；多模态请求在消息 `content` 中携带对应类型的 part（与 OpenAI 兼容结构一致）。
:::
