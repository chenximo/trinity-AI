# 速率与限额

Trinity 网关与账户维度设有请求速率、并发及配额限制，具体数值**以控制台展示与商务约定为准**。本文档为占位结构，对齐 [OpenRouter Limits](https://openrouter.ai/docs/limits) 阅读路径。

## 速率限制

| 类型 | 说明 |
| --- | --- |
| 每分钟请求数 | 超过可能返回 **429** |
| 并发连接 | 流式 SSE 占用连接数上限 |
| 日请求上限 | 与账户等级相关 |

::: info 待产品填写
请在发布前补充：默认免费档 / 付费档 QPS、日限额表及响应头（如 `Retry-After`）说明。
:::

## 配额与余额

- 余额不足或配额耗尽时，可能返回 **402** 或 **403**（以实现为准）。
- 充值或升级套餐后限额自动提升（以产品规则为准）。

## 客户端建议

1. 收到 **429** 时使用指数退避重试。
2. 记录 `x-request-id`（若响应提供）便于排查。
3. 生产环境对重试设置上限，避免放大流量。
