# 管理 API 密钥

在 Trinity 控制台创建、轮换与吊销 API 密钥，用于调用 HTTP API。

## 创建密钥

1. 打开 [Trinity 控制台 · API 密钥](https://trinitydesk.ai/account/keys)。
2. 点击 **创建密钥**。
3. 复制并妥善保存完整密钥（**仅展示一次**，前缀一般为 `xh-...`）。

::: tip 安全
勿将密钥写入前端代码、公开仓库或客户端日志。生产环境建议使用环境变量或密钥管理服务。
:::

## 轮换与吊销

| 操作 | 说明 |
| --- | --- |
| 轮换 | 创建新密钥后，在应用中切换并观察流量，再吊销旧密钥 |
| 吊销 | 立即失效，已发出的请求若仍携带旧密钥将返回 `401` |

::: tip 密钥泄露
密钥泄露时请先**吊销**再在控制台审计近期调用记录。
:::
