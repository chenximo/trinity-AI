## 2. 功能需求

### 2.1 KPI

| 指标 | 说明 |
|------|------|
| 我方 SOA 7d | 全启用问题均值 |
| 监测竞品数 | 链竞品管理 |
| 落后 / 领先题数 | 竞品 SOA vs 我方按题比较 |

### 2.2 竞品 SOA 排名

全问题集均值条形图；点击进 **竞品详情**。

### 2.3 同题 SOA 矩阵

行 = `question_id`；列 = 我方 + 各启用竞品；单元格 = 该题 7d SOA。

### 2.4 最大差距列表

落后幅度最大的题；链关键词详情 / 回答详情 / 竞品详情。

### 2.5 市场筛选

全部 / 海外 / 国内（与总览 Tab 一致）。
