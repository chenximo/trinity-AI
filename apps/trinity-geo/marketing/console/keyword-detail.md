## 1. 背景与问题

### 1.1 用户原声

> 「品类词 SOA 是 0，是豆包没提我还是所有平台都没提？」  
> 「这条问法要不要优先做内容？」

### 1.2 页面定位

| 项 | 说明 |
|----|------|
| **用户问题** | 「这道题我的可见性怎么样？」 |
| **产品角色** | 单 `question_id` 的 SOA 趋势 + 分平台摘要 + 最新回答入口 |
| **样本** | Q00「推荐两款 API 聚合平台」— **D1 品类失声**（SOA 0%，竞品进答案） |

## 2. 用户故事

```
作为 增长负责人，

以便 判断是全局失声还是个别平台问题，并点开具体回答取证。
```

## 3. 功能需求

### 3.1 页头

| 元素 | 说明 |
|------|------|
| 面包屑 | 监测 → 问题集 → Q00 |
| 问法全文 | 与问题集一致 |
| 类型标签 | 品类词 / 品牌词 / 对比词 / 场景词 |
| 状态 | 监测中 / 已暂停 |
| CTA | 返回问题集；查看最新回答 |

### 3.2 诊断横幅（条件展示）

当该题触发 D1–D4 时展示摘要条，链诊断模块（批 4）。

**Q00 样本**：D1 + S1+S2+S3 — SOA 0%；豆包 R1 信源盘 **0/16**。

### 3.3 KPI 行

| 指标 | 说明 |
|------|------|
| SOA 7d | 该题近 7 日进正文占比 |
| 信源盘 | 可提取平台最近一轮我方域 M/N（如豆包 R1） |
| CCR | 被引为信源占比 |
| 提及率 | 品牌名出现（含未进正文） |

采集样本量移至 KPI 行下方提示文案。

### 3.4 SOA 趋势

| 能力 | 说明 |
|------|------|
| 周期 | 日 / 周 / 月 |
| 数据 | `metric_snapshots` 按 `question_id` 过滤 |

### 3.5 分平台摘要表

| 列 | 说明 |
|----|------|
| 平台 | 10 平台之一 |
| 市场 | 国内 / 海外 |
| SOA 7d | 该平台该题 SOA |
| 信源 M/N | 该平台最近可提取回答的我方域命中（无则 —） |
| 竞品领先 | 进正文竞品名（若有） |
| 最近回答 | 时间 + 链回答详情 |

### 3.6 本题最新回答

列表：平台 · 轮次 · 日期 · 摘要 · 进答案徽章 · 诊断标记。
