## 1. 我们收集的个人信息

「**个人信息**」是指以电子或者其他方式记录的与已识别或者可识别的自然人有关的各种信息，不包括匿名化处理后的信息。

我们收集个人信息的情形包括：**您主动提供**、**您使用服务时自动产生**，以及**从第三方合法获取**（经您授权或法律允许）。

### 1.1 您主动提供的信息

可能包括：

- **账号注册与登录**：电子邮箱、用户名、密码或凭证；若您使用第三方账号登录（如 Google、GitHub 等，**实际上线渠道待运营确认**），我们会从该第三方获取其依据其政策向您披露的信息（例如公开资料、用户标识、电子邮箱）。
- **资料与表单**：您填写个人资料、工单、反馈、调研问卷时提供的信息。
- **API 与产品使用**：您通过本平台向**上游人工智能模型服务商**发送的提示词、对话内容、附件及配置参数（统称「**输入内容**」），以及模型返回的结果（「**输出内容**」）。输入、输出是否落库、保留多久及是否用于模型训练，见下文 **1.3** 与《服务条款》「用户内容」一节。
- **交易与商务**：购买额度/套餐、开具发票、企业客户签约时提供的联系人、公司名称、税号、对公账户信息等（**具体字段待运营确认**）。
- **沟通记录**：您联系客服或我们回复时的通信内容与联系方式。

### 1.2 自动收集的信息

当您访问网站或使用服务时，我们可能自动收集：

- **设备与日志**：IP 地址、浏览器类型与版本、操作系统、时区、设备标识符、访问时间、引荐来源、页面浏览与点击路径、搜索查询、API 调用日志（含错误码、延迟等，**日志字段粒度待运营确认**）。
- **Cookie 及类似技术**：见第 1.4 节。
- **安全与风控**：用于检测滥用、欺诈、攻击或违反《服务条款》的信号（例如异常调用频率、可疑登录）。

### 1.3 输入内容与输出内容（重要说明）

Trinity 提供 **AI API 聚合** 能力：您的请求可能经我们的网关**转发至第三方模型服务商**处理。我们**不控制**第三方对您输入、输出的后续使用（包括是否用于其模型训练）；其规则以该服务商条款为准。

### 1.4 Cookie 与同类跟踪技术

我们使用 Cookie、本地存储、像素标签及类似技术（统称「**跟踪技术**」），用于：

| 类型 | 目的 | 是否可关闭 |
|------|------|------------|
| **严格必要** | 维持登录态、安全、负载均衡、表单提交 | 通常不可关闭，否则服务无法正常运行 |
| **功能/偏好** | 记住语言、主题、界面偏好 | 可在浏览器中限制；部分功能可能受限 |
| **分析/统计** | 了解产品使用情况、改进体验 | **是否启用及具体工具待运营确认** |

您可通过浏览器设置管理 Cookie。禁用 Cookie 可能导致无法登录或使用部分功能。更多说明见【待补充：Cookie 政策链接，若有】。

## 2. 我们如何使用个人信息

我们仅在具有合法基础的情形下使用您的个人信息，目的包括：

- **提供与维护服务**：账号管理、API 鉴权与限流、路由至上游模型、Chat 与控制台功能、客户支持。
- **计费与履约**：额度/套餐购买、用量计量、账单与发票（**支付方式与机构待运营确认**）。
- **安全**：反欺诈、滥用检测、系统完整性保护、调查违反《服务条款》的行为。
- **改进产品**：统计分析、故障排查、功能开发（**若使用去标识化或聚合数据，见 2.1**）。
- **沟通**：服务通知、政策变更、与您请求相关的事务性邮件；**经您同意或法律允许的营销信息**（您可随时退订，见第 4 节）。
- **合规**：履行法律法规义务、响应监管或司法机关要求、主张或抗辩法律请求。
- **其他经您同意的目的**，或本政策披露时您可合理预期的目的。

### 2.1 聚合与去标识化信息

我们可能对个人信息进行去标识化或聚合处理，使其无法直接识别特定个人，用于：

- 生成用量、性能或趋势类统计报告（例如在文档或对内运营报表中展示，**对外公开范围待运营确认**）；
- 改进路由、定价与模型推荐策略；
- 在获得您单独同意时，与合作伙伴分享**无法识别个人身份**的统计结果。

去标识化信息在法律要求下仍可能重新识别，仅限必要且受控场景（如安全调查）。

## 3. 我们如何共享与披露个人信息

我们不会出售您的个人信息（「出售」按适用法律定义解释，**包括加州 CPRA 等场景待法务确认**）。

我们可能在下列情形共享个人信息：

| 接收方类型 | 目的 | 保障 |
|------------|------|------|
| **服务提供商** | 托管、CDN、邮件、支付、客服、安全、分析（**清单待运营确认**） | 合同约定保密与目的限制，不得用于自身营销 |
| **上游模型服务商** | 处理您的 API/Chat 请求 | 按其服务条款处理；我们仅传输提供服务所必需的数据 |
| **关联公司** | 统一账号、联合运营（**是否存在关联主体待运营确认**） | 受本政策约束 |
| **企业客户（B2B）** | 若您通过组织管理员开通账号，管理员可能查看约定范围内的用量与成员信息 | **以企业协议为准，待运营确认** |
| **经您同意** | 您明确授权的其他场景 | — |
| **法律要求** | 诉讼、调查、国家安全或执法要求等 | 在法律允许范围内可能无法事先通知您 |

**合并、收购或资产转让**：若我们涉及合并、分立、解散或资产转让，您的个人信息可能作为交易资产转移；我们将要求继受方继续受本政策约束或重新征得您的同意（**具体流程待运营确认**）。

## 5. 信息安全

我们采取合理的技术与管理措施（如访问控制、传输加密、安全审计等）保护个人信息，降低未经授权访问、披露、篡改或毁损的风险。

- 互联网传输无法保证绝对安全；请您妥善保管密码与 API Key，**勿与他人共享**。
- 发生可能影响您权益的安全事件时，我们将按法律法规要求向您告知并采取补救措施（**事件通报流程待运营确认**）。

::: caution 待运营确认
是否对外宣称已通过等保、ISO 27001、SOC 2 等认证：**仅可在取得证书后写入**。
:::

## 7. 个人信息保存期限

我们仅在实现本政策所述目的所必需的期限内保留个人信息，包括：

- 账号存续期间；
