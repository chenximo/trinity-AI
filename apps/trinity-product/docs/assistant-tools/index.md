# 智能助手工具 · 总览

> **定位**：Trinity **内部效能**向的智能助手与自动化工具集合（非对外产品线、不走完整 L0/L1/L2 手册树）。  
> **用户**：PM、技术支持、运营等内部角色。  
> **原则**：轻量、可狗食、真源优先落在 **钉钉文档 / 多维表**、Git、现有手册字段，**不为单个工具单独建业务库**。

---

| 工具 | 一句话 | 状态 | 文档 |

| [需求分析助手](./requirement-inbox/) | 钉钉群 `@整理` → 结构化候选 → 需求收件箱（多维表） | 路线 A 落地中（钉钉） | [产品需求](./requirement-inbox/prd) · [工程落地](./requirement-inbox/route-a) |

## 文档约定

```text
assistant-tools/
  index.md                 # 本页 · 工具总览
  <tool-slug>/
    index.md               # 工具简介与导航
    prd.md                 # 产品需求分析
    route-a.md             # 工程落地（或其它实现路线）
```

工程代码统一放在仓库 `trinity-AI/tools/<tool-slug>/`，在对应 `route-*.md` 中链出。
