# 用户控制台

> **工程**：`apps/ai-cloud/src/views/account/ConsolePage.vue` · 路由 `account/console`  
> **体验 / 在线**：见 [AI 云总览](./)

## 说明

注册/登录后的 **企业自助台**：查看多云子账号、费用、发票与联系顾问；四面板通过 hash 切换（`#accounts` · `#billing` · `#invoices` · `#contact`）。当前数据为 mock，**5.30 商用** 依赖运营后台与 API 联调。
