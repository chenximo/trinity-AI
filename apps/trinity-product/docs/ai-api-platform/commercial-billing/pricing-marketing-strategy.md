## 1. 与商务 / 策略的边界 {#boundary}

| 层 | 真源 | 营销页 |
|----|------|--------|
| 结算 | 预充值 Credits → 按量扣 | ✅ 主叙事 |
| 商务深折（如 0.65 族 9.0～7.2） | [证据链](./pricing-strategy-evidence-chain) · [Excel](./商务洽谈折扣总表.xlsx) | ❌ 不展示数字表 |
| 公开浅阶梯 | 对内折数不进页；页上用 **最高可省 %**（10 / 18 / 28） | ✅ 充值档位四卡 |
| 点名模型 | 洽谈表已启用 ID（优先 0.65） | ✅ 场景 / 爆款 / 示意价链目录 |
| 薄利 / 不设阶梯（如 0.97 seedance） | 矩阵「不设阶梯」 | ✅ 可作抢手/能力位；❌ 不当「最大折扣」 |
| 年框 / 更深合同 | 销售 | ✅ 仅 CTA「联系商务」 |

**改数顺序（强制）：** 证据链 / Excel → 本稿文案中的门槛与浅折 → 官网 `/pricing`。  
本稿 **不改** 商务主折。

## 2. 页面信息架构（营销页） {#page-ia}

阅读顺序：

```text
① 怎么开始（试用 · 按量 · 充值越多越优惠）+ **老带新横条**
② 限时特惠（爆款三席 · 有活动则显示）
③ 场景精选（Playlist）
④ 充值档位（预付费四档卡 · 「最高可省」不写折数）
⑤ 底部 CTA（模型目录 · 联系商务）
```

> **新客 / 老客** 不再单独成节：分别落在 ① 试用步与 ①③④ 充值叙事；**老带新**挂在计费区后，避免重复。

可选：① 后或 ③ 前放 **1～2 条价格示意**（链模型目录）。

### 2.1 ① 怎么开始（预付费 1·2·3）

| 步 | 标题 | 正文要点 | CTA |
|----|------|----------|-----|
| 1 | 免费试用 | 联系商务开通试用额度 · 用尽即停 | 联系商务 |
| 2 | 按量计费 | 按实际用量与模型刊例及商务折扣计费 | 联系商务 |

### 2.4 老带新（挂在 ① 后，不单开「新客·老客」节）

| 项 | 内容 | 状态 |
|----|------|:----:|
| **新客** | 见 ① 免费试用（示意 Trial **$3 / 30 天**） | 额度 [试用方案](./new-user-trial-scheme) **待拍** |

不做：在网时长自动折、新客永久另一套刊例；不做与 ① 重复的「新客/老客」三列。

### 2.5 ⑤ 充值档位（合并原「用量优惠 + 档位对比」）

| 档（中文） | 门槛文案 | 页上展示 |
|------------|----------|----------|
| 标准版 | 充值 &lt; $1,000 | 起步 · 刊例计价 |
| 进阶版 | 充值 ≥ $1,000 | 商务优惠 · **最高可省约 10%**（对齐 9.0） |
| 成长版 | 充值 ≥ **$10,000** | 商务优惠 · **最高可省约 18%**（对齐 8.2） |
| 企业版 | 充值 ≥ **$100,000** | 专属方案 · **最高可省约 28%**（对齐 7.2） |

- **不写具体折数**（9.0 / 8.2 / 7.2 只作对内换算）；中间档 $5k·15%、$50k·24% 页上不单列，留给商务。  
- **「最高可省」多杠杆**：商务方案 + 模型选型 + **节约成本插件** 等，脚注写明非固定折数。  
- **不再单独做「档位对比」表。**  
- 营销亮点：② 爆款可写限时折；本区只挂「最高可省」阶梯。

## 4. 爆款 / 限时特惠方法论 {#hit-products}

### 4.1 方向结论（已认）

选择 **三维三席** **没有问题**，且与经典「引流款 / 畅销款 / 形象款」同构：

| 席位 | 角色 | 对外标签 | 经济约束 |
|------|------|----------|----------|
| **A** | 引流款 | 最大优惠感 | 仅厚利可折族（优先 **0.65**）；公开浅惠，不露商务深折 |
| **B** | 畅销款 | 最流行 | 高认知 / 高调用；折不折独立判断 |
| **C** | 形象款 | 最抢手 | 可薄利（如 **seedance-2.0**）；**禁止**写成「最大折扣」 |

### 4.2 通用维度（需求 × 供给 × 经济 × 传播）

选任一爆款前用四组筛（或否决）：

| 组 | 看什么 |
|----|--------|
| **需求** | 流行度、场景是否一句话说清、痛点/稀缺 |
| **供给** | 上架稳定、容量、可改价可下架、有截止日期 |
| **经济** | 成本折族、能否进浅折/活动、客单与套利 |
| **传播** | 名字认知、故事类型（最便宜/最强/最新）、时效窗口 |

**否决规则：**

- 进 **A 折扣席**：成本族过薄（如 **0.97**）→ 否  
- 与试用强相关：不在试用白名单 → 慎（或 CTA 改充值）  
- 进 **C 抢手席**：可以薄利，文案不得承诺大折  

### 4.3 选爆款流程

```text
① 定本期目标（获客 / 首充 / 品牌话题）
② 用 §4.2 打分或否决
③ 每席 1 个主 SKU + 1 个替补
④ 定一句故事 + 截止日 + CTA
⑤ 上线后看：点击 → 试用或首充 → 该模调用占比 → 轮换
```

### 4.4 第一期三席示意（真模型 · 对齐洽谈表）

| 席位 | 候选 | 标签文案示意 |
|------|------|----------------|
| A 更省（折扣感） | **主推展示「GPT 系列」** + 举例 `gpt-5.3-codex · gpt-5.4 · gpt-4o`（无 5.5）· 落地价链 Codex | **限时 7.2 折**（约省 28%）· 联系商务开通 |
| B 更热（人气） | **主推** `deepseek-v4-flash`（0.70）· 替补 `gpt-4o` | **限时 8.8 折**（0.70 族勿硬套 7.2） |
| C 更抢手 | `seedance-2.0`（0.97） | 生视频现货 · **9.9 折**浅让（不套 7.2） |

> **官网已落地（2026-07-16）：** `/pricing` 限时特惠 = gpt-5.3-codex 7.2 · deepseek-v4-flash 8.8 · seedance-2.0 9.9；开通 CTA → Contact。

运营可按期替换，**角色席位固定**。同折（0.65·7.2）海外候选优先：**Codex（场景尖）> 4o（品牌宽）> 5.4-mini（单价更亲）**；国产绝对价锚仍用 qwen，不必占爆款席。
