# 用量门槛费率卡 v1（企业户 · 门槛 Q 测算）

> **文档类型**：商用计费 · **正式费率卡草案** + 单户月利测算（按门槛 \(Q\)）。  
> **读者**：产品、商务、财务。  
> **关联**：[定价场景方案卡](./pricing-scenarios-scheme) · [定价验证表 v1](./pricing-validation-v1) · **[定价策略与证据链（商务骨架真源）](./pricing-strategy-evidence-chain)** · [折扣差价专文](./discount-spread-tier-pricing) · [供应商成本折](../pricing-sources/supplier-cost-discounts)  
> **状态**：**草案 · 细档早期扫描底稿（部分过时）**  
> **定稿真源**：**[定价策略与证据链](./pricing-strategy-evidence-chain)**（Scale 对外 **7.6** · Scale+ **≥$100k·7.2** · 峰值测评过程见该文 **§2.2.1**）  
> **日期**：2026-07-14；状态注记 2026-07-15

---

## 0. 一句话

对 **6.5 成本族**、可折模型：按 **企业账户（一 Key / 一团队）月总消耗** 达档给对客折。  
测算只用 **门槛 \(Q\)**（该档门槛金额当作该户典型月刊例消耗），算单户月毛利；**不做**全盘进客假数池。

---

## 1. 口径（必读）

| 项 | 约定 |
|----|------|
| 客户主体 | **ToB 企业账户**（多人共用同一 Key / 工作区） |
| 达档指标 | **该户月刊例等价总消耗**（美元），不是个人、不是闲置预存 |
| 成本 | 商务原底 **6.5 折**，\(d_{cost}=0.65\)（刊例≈挂牌时） |
| 门槛 \(Q\) | 取该档门槛金额，作为「刚达档」典型单户用量 |
| 单户月毛利 | \(\Pi=(d_{sell}-0.65)×Q\) |
| 单位 GM | \(GM=1-0.65/d_{sell}\) |
| 月收入 / 月成本 | \(R=d_{sell}×Q\)，\(C=0.65×Q\) |

**本版不算 / 不依赖：**

- 全公司池子 \(Q\)（如 52k→505k 进客假数）
- 调用次数换算（可选附录，不影响定折）
- 「同 \(Q=50k\)、只改折」伪峰值

**Scale 深折**必须配套 **更高门槛**（本版：50k→7.5 / 60k→7.3 / 70k→7.0）。

### 3.1 峰值结论

| 项 | 结果 |
|----|------|
| **峰值档** | **≥ $50,000 · 7.5 折** |
| **最大单户月毛利** | **$5,000** |

| ≥$70k · 7.0 | \(\Pi=\$3,500\)（量 +40%，利 −30%，**偏激进**；宜再抬门槛或改浅折） |
