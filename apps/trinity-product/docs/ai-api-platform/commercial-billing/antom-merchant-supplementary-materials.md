## 2. Business Description（英文，审核主文）

**Company & Product**

Trinity AI Inc. operates **TrinityDesk** (https://trinitydesk.ai), a **B2B AI API aggregation platform** for developers and businesses. We provide a unified API and web console that routes customer requests to third-party large language model providers. TrinityDesk is an **independent platform** and is **not affiliated with, endorsed by, or sponsored by** OpenAI, Anthropic, Google, or other model providers.

**Target Customers**

- Software developers and engineering teams integrating LLM capabilities into applications
- Startups and enterprises building AI-powered products
- B2B customers purchasing pre-paid API credits for production or development use

We do **not** market TrinityDesk as a consumer chatbot product, adult content service, or uncensored model marketplace.

**How Customers Pay & Consume**

- Customers purchase USD **pre-paid Credits** (minimum USD $10 per transaction).
- Unless otherwise shown at checkout, customers receive Credits equal to the amount paid (e.g., pay USD $10, receive USD $10 in Credits).
- Credits are consumed only after **successful** API or chat requests, based on published per-model usage pricing.
- Credits remain valid indefinitely; there is no withdrawal or cash-out.
- Refunds: unused Credits from a purchase may be requested within **24 hours** via email; see Refund Policy at https://trinitydesk.ai/legal/terms#refund.

**Compliance & Abuse Prevention**

- **Acceptable Use Policy**: https://trinitydesk.ai/legal/acceptable-use  
  Prohibits sexually explicit content, violence, deepfakes, jailbreak/safety bypass, circumvention of rate limits, scraping/reselling API access, malware, and other abusive conduct.
- **Terms of Service** §3 summarizes acceptable use and references the full AUP.
- We may throttle, suspend, or terminate accounts and API keys for policy violations.
- Violations may be reported to TrinityAIInc@outlook.com.

**Data & Privacy**

- Privacy Policy: https://trinitydesk.ai/legal/privacy  
- Covers account data, API usage/billing logs, payment data processed via payment partners, and routing to third-party model providers.

**Support**

- Email: TrinityAIInc@outlook.com  
- Typical response: 1–2 business days

## 3. 业务描述（中文，内部对照）

**公司与产品**

Trinity AI Inc. 运营 **TrinityDesk**（https://trinitydesk.ai），面向开发者与企业的 **B2B AI API 聚合平台**。提供统一 API 与控制台，将客户请求路由至第三方大语言模型服务商。本平台为**独立第三方服务**，**与 OpenAI、Anthropic、Google 等模型厂商无隶属、背书或赞助关系**。

**目标客户**
