## 1. Excel 页签约定（不要改职责）

| Sheet | 职责 | 后续怎么扩 |
|-------|------|------------|
| `00_说明` | 文件说明、档名、页签索引 | 脚本重写 |
| `01_商务洽谈总表` | **一行一成本折**：阶梯折（含 GM）+ 模型清单格 | 新折加一行 + `src_xxx` |
| `02_交叉模型` | **宽表**：跨成本折同名 ID；线路格含 **优先级/权重**；推荐成本折 | 脚本按全量源自动重算 |
| `src_065` / `src_070` … | 对应导出「线路管理」**整表拷贝** | 新折新增 `src_0xx` |

**主表不做：** 优先级/权重、交叉模型明细（只在表末指向 `02`）。

## 3. 新增一个成本折（标准步骤）

### Step A · 导出源文件

1. 线路管理按 **目标折扣**（如 `8折`）+ **启用** 导出。  
2. 建议文件名：`model-supply-routes-YYYYMMDD (n).xlsx`，sheet 名保持 **`线路管理`**。  
3. 必备列：`模型编码` · `线路名` · `优先级` · `权重` · `折扣` · `启停`（与现导出一致）。

### Step B · 折数是否已在证据链？

| 情况 | 动作 |
|------|------|
| 已有族（0.60～0.90 / 0.97…）且阶梯已拍 | 只回灌模型，**不改** `FAMILY_TIERS` 折数 |
| **新**成本折 / 要改阶梯折 | **先改** [证据链](./pricing-strategy-evidence-chain) §3 / §5 → 再改脚本 `FAMILY_TIERS` |

### Step C · 登记脚本 `SOURCES`

在 `scripts/rebuild_discount_tier_workbook.py` 的 `SOURCES` 增加一行：

```python
# (本地导出路径, 折扣列原文, d_cost键, src sheet 名)
(Path(".../model-supply-routes-….xlsx"), "8折", "0.80", "src_080"),
```

并把该族加入 `FAMILY_TIERS`（五档对客折 + 操作带 + 是否进公开阶梯）。

### Step D · 跑脚本

```bash
cd trinity-AI/apps/trinity-product/docs/ai-api-platform/commercial-billing
python3 scripts/rebuild_discount_tier_workbook.py
```

默认写出同目录 `商务洽谈折扣总表.xlsx`。

### Step E · 抽查清单

- [ ] `01`：新成本折行有模型数、模型格 `ID：线路；`+换行、档位格含 `GM`  
- [ ] `src_0xx`：原表列齐全  
- [ ] `02`：若新折与旧折出现同名 ID，宽表自动多出交叉行；线路格为 `线路 · P1/100`  
- [ ] 推荐成本折仍遵循：**数字越小越优先**（0.65 > 0.70 > …）  
- [ ] MD：`discount-tier-matrix.md` 补一行「已导入」摘要（可选）

## 4. 口径速查（写进表的不要偏离）

| 项 | 口径 |
|----|------|
| 达档 | 企业户**月消耗**（刊例）· **非预存** |
| 适用范围 | 商务合同 / 销售带内（公开阶梯另文） |
| 单位 GM | \(GM=1-d_{cost}/d\)，写在档位格内，如 `9.0（GM 28%）` |
| ≥0.95（含 0.97） | **不设用量阶梯**；对客原价 |
| 交叉推荐 | 更厚利成本折优先；P/W 仅作默认路由参考，**不是**商务折依据 |
| 乱码线路名 | 导出若出现 `??????`，脚本替换为 `网聚云联`（已知 (4).xlsx 问题） |

## 5. 什么时候改文档、什么时候只改 Excel

| 变更 | 证据链 | 脚本 FAMILY_TIERS | 仅重跑脚本 |
|------|--------|-------------------|------------|
| 新导出、同族补模型 | — | — | ✅ |
| 新成本族、阶梯已拍 | 若未写入则补 | ✅ 加行 | ✅ |
| 改 Plus/Mid… 折数 | ✅ 先改 | ✅ 同步 | ✅ |
| 改档名英文 | ✅ §3.0 | ✅ 表头常量 | ✅ |
| 公开营销阶梯 | **另文**，不进本 Excel | — | — |
