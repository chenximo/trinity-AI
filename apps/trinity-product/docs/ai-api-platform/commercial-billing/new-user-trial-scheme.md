## 0. 一句话

业界用「免费额度 / 限流免费模型」获客；Trinity 用 **小额 Trial Credits + 上游厚利模型白名单 + 用完即停**，成本受 [商务原底](../pricing-sources/supplier-cost-discounts) 约束，付费后再进入 [证据链](./pricing-strategy-evidence-chain) 用量阶梯。

## 2. 业界做法整理

> 信息截至调研日（2026-07）；以各厂文档/公告为准，变更时改本节。

### 2.1 总览对照

| 维度 | 阿里云百炼 | 腾讯云 TokenHub | OpenRouter | 其他常见形态 |
|------|------------|-----------------|------------|--------------|
| **体验形态** | 分模型免费额度（推理） | 分模型/品类免费包（Token 或次数/积分） | **:free 模型** + 严格限流；少送现金 Credits | 试用金 $1～$30；或厂商自送百万 token |
| **额度量级** | 各模型不同（控制台展示） | 语言模型约 **100 万 Tokens**；视觉另计次数/积分 | Token **$0**，靠 **次/日限额** | DeepSeek 等偶见千万级 token 试用 |
| **有效期** | 约 **30～90 天**（新人额度自 2025-09 起多为 **90 天**） | 语言/多模态理解 **90 天**；部分视觉 **365 天** | Credits 常不过期；免费档是限流不是赠金 | 7～90 天不等 |
| **领取** | 开通即发 | 模型广场勾选领取 / **首次调用自动领** | 注册即可打 free 模型 | 活动页 / 控制台 |
| **用尽后** | 可开「用完即停」；未认证常需认证+充值后才能按量 | **未开后付费则停**；开启后按量 | 仅能继续用 free 或需购 Credits | 停或自动转付费 |
| **抵扣顺序** | 免费 → 资源包 → 节省计划 → 按量 | 免费优先，再后付费 | free 模型不耗 Credits；付费模型耗余额 | — |
| **账号规则** | 账号/模型维度额度 | 主账号；每模型领一次；不可转让叠加兑现 | 账号级限流；充 ≥$10 抬高 free 日限额 | — |
| **付费包装（相关）** | 资源包、节省计划、Coding Plan 等 | Token Plan / Coding Plan / 企业轻享·专业 | Pay-as-you-go Credits + Enterprise | 包月 / CUD |

### 2.2 阿里云百炼（摘要）

| 项 | 内容 |
|----|------|
| 入口 | 首次开通百炼自动发放新人免费额度 |
| 用途 | 主要抵扣 **实时推理**；批量/微调/部署等常不在免费范围内（以文档为准） |
| 期限 | 新人额度有效期调整后多为 **90 天**（历史用户可能不同）；部分「选择模型」类为按月发放 |
| 防误扣 | 推荐开启 **免费额度用完即停**；未认证用户用尽后常无法继续，需认证并充值 |
| 付费衔接 | 抵扣优先级：免费额度 → 资源包 → 节省计划 → 按量付费 |
| 文档 | [新人免费额度](https://help.aliyun.com/zh/model-studio/new-free-quota) · [有效期调整通知](https://help.aliyun.com/zh/model-studio/new-free-quota-validity-adjustment) · [节省计划/资源包](https://help.aliyun.com/zh/model-studio/savings-plan-and-resource-package) |

**对 Trinity 的启示：** 「开通即有 + 用完可停 + 明确抵扣顺序」必须具备；期限 90 天偏松，可用更短期限促首充。

### 2.3 腾讯云 TokenHub（摘要）

| 项 | 内容 |
|----|------|
| 入口 | 模型广场「新用户福利」勾选领取；或支持模型 **首次调用自动领取** |
| 语言 / 多模态理解 | 约 **100 万 Tokens** / 模型向，**90 天** |
| 视觉等 | 图像次数、视频/3D 积分等，有效期可达 **365 天**（分产品） |
| 规则 | 主账号维度；每账号每模型一次；不可转让/叠加/兑现；过期作废 |
| 用尽 | 未开启后付费 → **服务停止**；开启后按 Token 计费 |
| 活动期 | 文档曾注明活动截至日期（如 2026-12-31），以页面为准 |
| 文档 | [新人免费体验包](https://cloud.tencent.com/document/product/1823/130053) · [计费方式](https://cloud.tencent.com/document/product/1823/130054) |

**对 Trinity 的启示：** 「百万 Tokens」心智强；可用 **$ 赠额 + 折算话术**（例：约等于 flash 上百万级 token）对外对齐，对内仍按 Credits 扣。自动领取降摩擦可作增强。

### 2.4 OpenRouter（摘要）

| 项 | 内容 |
|----|------|
| 体验形态 | **不主打试用金**；提供大量 **:free** 模型（$0/token） |
| 限流 | 约 **20 RPM**；未充值约 **50 次/日**；一次性购 Credits **≥$10** 后 free 日限额可升至约 **1000 次/日** |
| 付费 | 预充 Credits；购买侧常有平台费（公开材料多见约 5.5% 量级，以官网为准） |
| 文档 | [Pricing](https://openrouter.ai/pricing) |

**对 Trinity 的启示：** 聚合商可用「限流免费模型」控成本；Trinity P0 仍以 **Trial Credits** 为主（与现有钱包一致），**:free 模型池** 可作 P1。

### 2.5 其他常见做法（备忘）

| 形态 | 代表倾向 | 备注 |
|------|----------|------|
| 试用金 / 代金券 | 部分云与 API 平台 | $1～$30 居多；需防刷 |
| 厂商直连大额 token | 部分模型官网活动 | 难直接对标聚合多模型 |
| 包月场景套餐 | 百炼 Coding Plan、TokenHub Token/Coding Plan | 属「套餐包」而非纯试用；Trinity 档位套餐另文 |

### 2.6 业界共性（可复用原则）

1. **有上限**：额度或限流，二者必居其一（或并存）。  
2. **有期限或有日限额**：促使用与转化。  
3. **用尽默认可停**：防「试用变欠费」。  
4. **账号一次 / 防叠加兑现**。  
5. **体验与生产付费包装分离**：免费之后才是资源包、阶梯、包月。  
6. **贵模态（视频/图）额度更严或另包**。

## 3. Trinity 约束（从已有真源推导）

### 3.1 上游成本折 → 体验池准入

依据 [商务原底](../pricing-sources/supplier-cost-discounts) 与 [证据链 §2.1.1](./pricing-strategy-evidence-chain#cost-family-list-gm)：

| 成本折（台账） | 例子 | 刊例 GM（刊例≈挂牌） | 体验池 |
|----------------|------|----------------------|--------|
| **6 / 6.5** | 百炼 HappyHorse·6；qwen / 网聚 GPT·**6.5** | ~35%～40% | **主力白名单** |
| **7** | 百炼「其他」 | ~30% | 可进，额度更紧 |
| **7.5 / 7.8** | 网聚 Gemini·7.5；YM/南清 Claude·GPT·**7.8** | ~22%～25% | **慎进**（小配额或仅点名试） |
| **9.7** | 南清 seedance | ~3% | **默认不进** |
| 台账未列 | TokenHub / AIGC 等 | 待补 | 仅当后台生效折确认属厚利族后再进 |

### 3.2 对客阶梯 → 试用后衔接（不在试用内打折）

已拍（0.65 族主锚，摘要）：

| 档 | 门槛（企业户月耗） | 主折 |
|----|-------------------|------|
| Standard | &lt;$1k | 原价 |
| Plus | ≥$1k | 9.0 |
| Mid | ≥$5k | 8.5 |
| Growth | ≥$10k | 8.2 |
| Scale | ≥$50k | 7.6 |
| Enterprise | ≥$100k | 7.2 |

（档名见 [证据链 §3.0](./pricing-strategy-evidence-chain#tier-naming)）

试用阶段：**不发放阶梯折**；用尽文案可预告「月耗达标后享档位优惠」。

### 3.3 计费总览已有约定

见 [commercial-billing §4.4](./#44-试用与免费档待拍板)：不绑卡 + 低额度（待拍）；用尽停止 + 引导充值（建议）。

## 4. Trinity 新人体验方案（草案）

### 4.1 产品定义

| 项 | 草案 | 状态 |
|----|------|:----:|
| 名称 | **Trial Credits**（试用余额） | 草案 |
| 发放 | 注册成功自动到账；**一账号一次** | 草案 |
| 绑卡 | **不要求**（6.30） | 与总览一致 |
| 额度 | **$3**（备选 $2 / $5） | **待拍** |
| 有效期 | **30 天**（自到账日起；备选 14 / 90） | **待拍** |
| 用尽 / 到期 | **停止调用**（如 402）+ 控制台引导首充 | 建议锁定 |
| 科目 | 与充值 Credits 分账或可合并展示但 **来源标记 Trial** | 待工程 |
| 达档 | 试用消耗 **不计入** 企业户月耗达档 | **建议锁定** |

### 4.2 模型与路由

| 规则 | 草案 |
|------|------|
| 白名单 | 仅 **成本折 ≤6.5（系数 ≤0.65）且已上架、后台生效折已确认** 的生文模型（例：deepseek-v4-flash / pro、qwen 族、空间足够的 GLM 等——以运营清单为准） |
| Auto / 默认路由 | 试用期内 **只落白名单** |
| 点名非白名单 | 拒绝或提示「试用不支持，请充值后使用」 |
| 生图 / 生视频 / seedance 等 | **6.30 不进试用** |
| 7.8 / 薄利族 | 默认不进；若运营加白，须单独限更小有效额度 |

### 4.3 对外话术（示意）

- 「注册即送 **$3** 试用额度，约 **30 天**内有效。」  
- 「可用于平台精选高性价比模型（含 DeepSeek / Qwen 等）；用尽或到期后请充值继续。」  
- 「正式用量按月消耗享受阶梯优惠（详见定价）。」  
- 可选折算：「以 deepseek-v4-flash 为例，约相当于百万级 Tokens 量级（视缓存命中与调用方式而变）。」

### 4.4 用尽 / 将尽运营

| 触点 | 动作 |
|------|------|
| 余量 &lt;20% | 站内提示 + 推荐首充档（如 $20 / $50，刊例） |
| 用尽或到期 | 停止调用；CTA：Purchase Credits；次要 CTA：查看档位优惠说明 |
| 首充成功 | Trial 结束；后续走刊例 + 达档规则 |

### 4.5 防滥用（最低集）

| 手段 | 说明 |
|------|------|
| 一账号一次 | 与主账号 / 用户 ID 绑定 |
| 白名单 + Auto 限池 | 防止烧 Claude / 视频 |
| 速率限制 | 试用 Key 更严的 RPM/TPM（数待工程） |
| 设备 / 支付指纹 | P1；P0 靠额度小 |

### 4.6 与业界对标结论（我们怎么选）

| 业界要素 | Trinity 选择 | 理由 |
|----------|--------------|------|
| 分模型 token 包 | **改 $ Credits 池 + 白名单** | 与钱包/价目一致，易控多模型亏钱 |
| 90 天 | **先 30 天** | 促首充；验证后再放宽 |
| 百万 Tokens 话术 | **可作营销折算** | 对内仍 $ |
| 用完即停 | **默认停** | 对齐百炼/TokenHub + 总览 |
| 不绑卡 | **是** | 降摩擦；靠额度与白名单控成本 |
| free 限流模型 | **P1** | 不挡 P0 Credits 试用 |

### 4.7 粗算 CAC（示意 · 拍板后替换）

假设：额度 **$3**，用户把试用打在白名单、刊例≈挂牌、成本 6.5 折 → 平台成本约 \(3×0.65≈\$2.0\) / 用尽用户（忽略缓存增益）。  
若激活试用占比 α、用尽占比 β，则期望 CAC_试用 ≈ \(3×0.65×α×β\) 量级（再加支付与支持）。  
**正式财务以真实命中与路由混合成本复核。**

## 5. 明确不做（本阶段）

| 项 | 原因 |
|----|------|
| 试用期内发放 Plus/Mid 商务折 | 与阶梯达档混淆；体验应独立 |
| 试用含 seedance / 未确认薄利线 | 上游无空间 |
| 充值加赠与试用叠加成「又送又赠」大礼包 | 方案卡曾划出边界；易失控 |
| 对标百炼完整资源包矩阵 | 属付费包装，另文 |
| 加密货币领试用 | MVP 不做加密支付 |
