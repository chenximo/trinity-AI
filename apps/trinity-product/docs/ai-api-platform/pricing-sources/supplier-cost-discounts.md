# 模型原价折扣

> **关联**：[模型价格真源 · 总览](./) · [采购渠道真源](./suppliers) · [价目全流程](./workflow)  
> **读者**：商务、运营、产品  
> **维护约定**：**本页为供应商成本折扣的唯一商务原底**；合同变更只改 §商务原底。生效折扣在**后台管理系统**手动维护，不同步价目 pipeline。

## 是什么

**模型原价折扣** = 我们与各进货渠道签订的、相对其**挂牌原价**的成本折扣。

| 概念 | 说明 |
|------|------|
| **挂牌原价** | 供应商价目表 / 控制台标价（见 [采购渠道真源](./suppliers)） |
| **成本折扣** | 协议折扣；**本页台账用「X折」**（如 `6.5` = 6.5 折） |
| **成本价 / 毛利** | 在**后台管理系统**按生效折扣测算；**不在价目 Excel** |

## 维护分工

| 层级 | 位置 | 谁维护 | 用途 |
|------|------|--------|------|
| **商务原底** | 本页 §商务原底 | 商务 / 运营 | 合同折数、预付/后付、信控等备忘 |
| **生效折扣** | 后台管理系统 | 运营手动填写 | 线上实际生效 |
| **成本 / 毛利** | 后台管理系统 | 运营 / 财务 | 内部毛利测算 |
| **价目 Excel** | `trinity-pricing-*.xlsx` 供应商分表 | 自动生成 | **仅挂牌价、刊例对比**；不含折扣/成本列 |

与 [刊例对比主表](./workflow) 的关系：

- **刊例对比校验**：对 **挂牌价**，不含折扣。
- **供应商分表**：只展示挂牌价与 vs 官方；成本折扣与毛利见**本页 + 后台**。

## 折扣记法

本页台账用 **「X折」** 整数或一位小数（如 `6.5`、`7.8`）。录入后台时按系统要求换算（常见为系数 `X ÷ 10`，如 6.5 折 → `0.65`）。

| 台账（折） | 系数 | 含义 |
|------------|------|------|
| `10` / 无折扣 | `1.0` | 原价 |
| `8.5` | `0.85` | 85 折 |
| `7.8` | `0.78` | 78 折 |
| `6.5` | `0.65` | 65 折 |
| `6` | `0.60` | 60 折 |

## 变更流程

1. 商务更新合同或台账 Excel。
2. 运营更新 **本页 §商务原底**。
3. 运营在 **后台管理系统** 按本页折数手动录入（或调整）生效折扣；成本与毛利在后台测算。

价目 Excel **不含**折扣/成本列（pipeline 只管挂牌价与刊例对比）。
