## 5.30 能力门禁（草案）

| 优先级 | 模块 | 门禁 |
|--------|------|------|
| **P0** | 供应商 → 模型 → 平台密钥 | 可配置一条完整上游链路 |
| **P0** | 用量明细 | 可查 `usage_event` 聚合（原型接真 API 或准 live） |
| **P0** | 监控大盘 | 核心 KPI 接 live 或准 live |
| **P0** | 权限与审计 | 后台登录 + **写操作审计最小集** |
| P1 | 用户审核 | 单条通过/拒绝可走通 |
| P1 | 用户 Key 运营 | 冻结/限额接 API |
| P2 | 客户与合同 | mock 演示即可 |
| **不做 5.30** | 报表全量、§4.6 路由策略独立页 | 排商用 |

用例与 Bug：[飞书 · 5.30 体验测试表](https://qcn81yhei1l2.feishu.cn/sheets/PjnVs7bmphodaKtOkkycpvxmnne)。
