# trinity-design-tokens · 领域边界

## 本 Skill 管什么

- `assets/trinity-base.css` 全局 token
- `TrinityAI/design-tokens.html`、`design-spec.html`（或 `apps/trinity-design` 同源）
- `docs/01-原型与交付规范/Trinity版式与视觉规范.md`
- 规范先行：定稿前迭代规范页，不扩散改业务页

## 本 Skill 不管什么

| 不归本 Skill | 交给 |
|--------------|------|
| 控制台页头/列表规则 | `trinity-user-console` |
| 后台表格 | `trinity-admin-ruoyi-list` |
| 营销页模块叙事 | `trinity-tob-marketing-site` |
| Monorepo 包结构 | `trinity-vue-prototype-monorepo` |

## 持久规则

`.cursor/rules/trinity-design-spec-first.mdc`（alwaysApply）与本 Skill 叠加。
