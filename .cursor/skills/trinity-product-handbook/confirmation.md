# trinity-product-handbook · 确认规则

## 须先确认

| 操作 | 说明 |
|------|------|
| **删除手册页面** | 含叶子、总览、子总览 |
| **改进度真源** | 批量改 `roadmap.yml` 符号、锁节点、week-progress 结构 |
| **改侧栏 IA** | 移动/重命名整组导航 |
| **git 提交** | 提交手册 docs + yml 变更 |
| **落地工程** | 用户未明说时，不得改 `apps/trinity-ai` 等实现代码 |

## 不需要确认

- 单页文案修订（不改页型）
- 周会 routine 更新 `week-progress.yml` 的 `plan` / `blockers`（备注）行
- 修正链接 typo

## 确认话术

```text
将删除/改版 <path> 并更新侧栏，是否继续？
```
